# 🏛️ #Museum Website

A responsive museum web application built using modern frontend technologies.  
The project represents a dynamic museum interface with structured content, clean UI, and scalable architecture inspired by real-world digital museum platforms.

---

## 🚀 #Features
- Responsive layout (desktop, tablet, mobile)
- Structured museum content and sections
- Clean and modern UI design
- Reusable components and scalable architecture
- SCSS for maintainable styling
- Interactive elements and smooth UX
- Optimized layout for real-world applications

---

## 🛠️ #TechStack
- HTML5
- SCSS (Sass)
- JavaScript
- BEM methodology
- npm

---

## 🎯 #Purpose
This project was created to practice building real-world web interfaces, improve layout structuring skills, and simulate a modern museum website with scalable and maintainable frontend architecture.

## 🚀 #GettingStarted

### 📥 Clone the repository
git clone https://github.com/PolianskyiDmytro/Museum.git

cd Museum

### 📦 Install dependencies
npm install  
# or  
yarn install  

### ▶️ Run the project locally
npm start  
# or  
yarn start
